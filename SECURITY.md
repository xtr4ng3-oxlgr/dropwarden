# Security Policy

DROPWARDEN es una herramienta defensiva local.

No agregar:

- ejecución automática de muestras,
- eliminación de archivos,
- cuarentena automática,
- subida de hashes o archivos a servidores,
- explotación,
- fuerza bruta,
- módulos ofensivos.

La herramienta debe seguir siendo de lectura, análisis y reporte.
