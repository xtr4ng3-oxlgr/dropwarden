# Señales

DROPWARDEN usa señales de riesgo:

- archivo descargado de internet,
- ejecutable sin firma,
- firma inválida,
- doble extensión,
- nombre sospechoso,
- script con comandos sensibles,
- ZIP con ejecutables,
- alta entropía,
- muchas entradas de inicio,
- poca RAM o poco disco para instalar cosas pesadas.

Una señal no significa malware. Significa revisión.
